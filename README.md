# 🚀 Portable Log Analyzer

**Portable Log Analyzer** is a lightweight browser-based web application built using **React, Vite, and Tailwind CSS**.  
It allows users to easily upload, search, filter, and analyze log files locally without relying on heavy software or cloud platforms.

---

## 📌 Project Description

This tool helps cybersecurity students, SOC analysts, developers, and IT administrators quickly investigate log data for troubleshooting and security analysis. Users can upload `.log` or `.txt` files, filter entries by keywords or severity levels (info, warning, error), and highlight critical events to identify abnormal or suspicious behavior.

All processing happens directly in the browser, keeping sensitive log data private and secure while delivering fast results.

---

## 🛠️ Tech Stack

- **React** – Frontend framework
- **Vite** – Fast build tool and dev server
- **Tailwind CSS** – UI styling
- **JavaScript** – Log parsing and filtering logic

---

## ✨ Features

- Upload and display log files
- Real-time search across log entries
- Filter logs by severity level
- Highlight warnings and errors
- Clean & responsive interface
- Fully local processing (no data upload)
